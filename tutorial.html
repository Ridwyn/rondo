How to create HTML Forms

This tutorial provides a series of articles that will help you master HTML forms. HTML forms are a very powerful tool for interacting with users; however, for historical and technical reasons, it's not always obvious how to use them to their full potential. In this guide, I'll demontrate all aspects of HTML forms, from structure to styling, from data handling to custom widgets.

Prerequisites

Before starting this module, you should at least work through Introduction to HTML. At this point you should find the Basic guides easy to understand, and also be able to make use of our Native form widgets guide.

Native Form Widgets

Attribute Name Description
form The form element that the widget is associated with. The value of the attribute must be the id attribute of a <form> element in the same document. In theory, it lets you set a form widget outside of a <form > element. In practice, however, there is no browser which supports that feature.
name The name of the element; this is submitted with the form data.
value The element's initial value.
autofocus This Boolean attribute lets you specify that the element should automatically have input focus when the page loads, unless the user overrides it, for example by typing in a different control. Only one form-associated element in a document can have this attribute specified.
